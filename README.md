## 下载

### Docker

```
docker pull apachecn0/stats-thinking-21-zh
docker run -tid -p <port>:80 apachecn0/stats-thinking-21-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g stats-thinking-eryi-zh
stats-thinking-eryi-zh <port>
# 访问 http://localhost:{port} 查看文档
```
